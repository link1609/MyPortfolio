View Code This Calculator application is built using Python. The tkinter module is need for the application to work. As tkinter is used to design the GUI. Designed this project to familiarise myself with tkinter as tkinker can get pretty complicated but was simple enough for this project. On this project I created buttons for numbers, symbols, screen and body using tkinter then assigned them to their own functions.

I'm happy with this project as it works the way a calculator should with addition, subtraction, multiplication and division functions. I encountered some problems installing tkinter on linux so I could get this to work on Linux, your usage may vary.

View Code Simple Snake game, built in python. Pygame tracks key presses for movement. This application works using the pygame module. The snake gets bigger when eating apples and increases the score, collision detection is used to end the game if the snake gets tangled in itself. The game also has background audio as well as audio when eating an apple and on collision.

View Code MP3 player built in Python. The following Python modules are used in this project. Tkinter, pygame, mutagen.mp3, os and time. Tkinter is used to design the GUI for the application including screen, buttons, status bars as well as message boxes. Pygames mixer is used for audio playback and mutagen.mp3 is used to track the length of audio.

Contains working Play, Pause, Stop, Next Song, Previous song commands. Can also open a single song using file explorer adding it to the playlist or can add multiple songs and the playlist can also be deleted. Also contains a working seekbar allowing the user to skip to a desired part of a beloved song. The included Try and except boxes stops the application from crashing if there is no available song to play, when using previous or next buttons.
